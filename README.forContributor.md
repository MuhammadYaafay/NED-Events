# Event_Management_System


## Getting Started
To get started with the project, follow these steps:

### 1. Clone the Repository
Download a copy of the project to your local machine by running:
```bash
git clone https://github.com/MuhammadYaafay/Event_Management_System.git
```

### 2. Navigate to the Project Directory
Change the current working directory to the project's folder:
```bash
cd Event_Management_System
```

### 3. Push New Code Changes
If you want to push your code changes to the repository, follow these steps:

#### Step 1: Pull the Latest Changes from the Remote Repository
This ensures your local repository is up-to-date:
```bash
git pull
```

#### Step 2: Stage Your Changes
Add the specific file(s) you've modified:
```bash
git add <file-name>
```
To add all changed files, use:
```bash
git add .
```

#### Step 3: Commit Your Changes
Provide a clear and concise commit message describing your changes:
```bash
git commit -m "Description of the changes you made"
```

#### Step 4: Push Your Changes to the Remote Repository
```bash
git push
```
